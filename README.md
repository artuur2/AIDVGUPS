# Распознавание образов с помощью ИНС

Программа `pattern_recognition_nn.py` реализует сеть **с прямыми связями** и обучение методом **обратного распространения ошибки**.

## Что исследуется

Перебираются и сравниваются параметры:
- функция активации: `sigmoid`, `tanh`, `arctan`;
- количество скрытых слоев: `0`, `1`, `2`;
- норма обучения: `0.01`, `0.05`, `0.1`;
- число эпох: `1000`.

## Данные

Используется набор 5x7 шаблонов цифр `0-9`, из которых генерируются зашумленные образы.
Это позволяет получить воспроизводимый датасет для задачи распознавания образов.

## Запуск

```bash
python3 pattern_recognition_nn.py
```

После запуска выводится таблица точности для всех комбинаций параметров и автоматически выбираются оптимальные параметры сети.
