# Распознавание образов с графическим интерфейсом

Программа `pattern_recognition_nn.py` реализует:
- сеть **с прямыми связями**;
- обучение методом **обратного распространения ошибки**;
- **графический интерфейс** (Tkinter) для загрузки эталонов и распознавания новых изображений.

## Возможности

1. Загрузка своих эталонных изображений для обучения:
   - выберите папку, где каждая подпапка — это отдельный класс;
   - внутри подпапок разместите изображения (`png`, `jpg`, `jpeg`, `bmp`).
2. Настройка параметров сети:
   - функция активации: `sigmoid`, `tanh`, `arctan`;
   - скрытые слои (например, `64,32`, `32` или пусто для 0 скрытых слоёв);
   - норма обучения: `0.01`, `0.05`, `0.1`;
   - количество эпох (по умолчанию `1000`).
3. Распознавание:
   - загрузка произвольного изображения вручную;
   - распознавание случайного изображения из выбранной папки.

## Структура эталонного датасета

Пример:

```text
train_data/
  circle/
    c1.png
    c2.png
  square/
    s1.png
    s2.png
```

## Запуск

```bash
python3 -m pip install -r requirements.txt
python3 pattern_recognition_nn.py
```

## Примечание

Для устойчивой работы желательно, чтобы изображения эталонов и распознаваемые изображения были похожи по стилю (контраст, масштаб, фон).
